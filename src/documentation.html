Documentation

Introduction

This is the project that implements mazes and caves including the basic algorithms of their handling, such as: generation, rendering and solving.

Installation

Can be opened in QT creator or create the executable using make install.

Maze

Maze is the program that allows to generate "perfect" or "simply connected" with no loops mazes according to Eller's algorithm and find the shortest path using A* search algorithm.

Description

The app supports the following features.

  • load a maze from an .txt file that contain numbers of rows and columns and two matrices with the positions of vertical and horizontal walls
  • generate a perfect maze
  • save the generated maze in the .txt file format described above automatically
  • solve a maze
How to use
  1. To load the maze, click "Load froam file..." and select the required file.
  2. To generate a maze, click "Generate".
  3. To change the dimensions of the generated maze enter the desired values in the corresponding windows.
  4. To solve the maze:
    • select the first point by clicking with the left mouse button
    • select the second point by clicking with the left mouse button
    • deselect the first point or delete the drawn path with the right mouse button

Cave

Cave is the program that allows to generate cave using a cellular automaton.

Description

The app supports the following features.

  • load a cave from an .txt file that contain numbers of rows and columns and matrix with the positions of "live" and "dead" cells
  • generate a cave
  • rendering the next step in two variants: by pressing the next step button or by pressing the start automatic work button that starts rendering iterations of the algorithm with a frequency of 1 step in N milliseconds
How to use
  1. To load the cave, click "Load froam file..." and select the required file.
  2. To generate a cave, click "Generate".
  3. To change cave's generation/rendering parameters enter the desired values in the corresponding windows.
